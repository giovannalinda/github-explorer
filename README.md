## Getting Started
### Install dependencies:

```bash
yarn
```

or

```bash
npm install
```

### Run development server

```bash
yarn dev
```

or 

```bash
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) with your browser to see the result.

## Commands

- `dev`: run development server
